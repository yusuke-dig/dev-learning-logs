## 今日学んだこと

### forEach と $0

```swift
numbers.forEach { result += $0 }
```

`$0` はクロージャの第1引数の省略記法。`forEach` は各要素に対して処理を実行する。

### filter でカウント

```swift
array.filter { $0 == target }.count
```

条件に一致する要素だけを抽出してから `.count` で個数を得るパターン。

### Dictionary で頻度カウント

```swift
var freq: [Int: Int] = [:]
array.forEach {
    freq[$0] = (freq[$0] ?? 0) + 1
}
```

キーが存在しない場合は `?? 0` でデフォルト値を補う定番イディオム。

### Dictionary.max(by:) で最大値エントリを取得

```swift
freq.max(by: { $0.value < $1.value })?.key
```

`Dictionary` の `max(by:)` は `(key, value)` のタプルを返す。`.key` で最頻値の要素を取り出せる。
