## 今日学んだこと

### Swiftの4つの基本型

Swiftには `Int`（整数）、`Double`（小数）、`String`（文字列）、`Bool`（真偽値）という主要な基本型がある。
型推論が強力で、右辺の値から型を自動決定してくれる。

```swift
let age = 20         // Int と推論
let pi = 3.14        // Double と推論
let name = "Alice"   // String と推論
let flag = true      // Bool と推論
```

型を明示する場合でも推論と組み合わせてよい。コンパイラが整合性を保証してくれる。

---

### 型変換は明示的に行う

Swiftでは異なる型同士の演算を暗黙的に行わない。`Int` と `Double` を混在させると型エラーになる。

```swift
let n: Int = 5
let d: Double = 1.5
// let result = n + d  // エラー: 型が合わない

let result = Double(n) + d  // OK: 明示的に変換
```

この設計により、型の不一致によるバグが早期に発見できる。

---

### String.count はUnicode文字単位で数える

`String.count` は絵文字や合成文字も「1文字」として正しく数える。これはSwiftの `Character` 型がUnicodeクラスターを1単位として扱うため。バイト数ではなく意味的な文字数を返す点が重要。

```swift
"hello".count   // 5
"swift".count   // 5
"kotlin".count  // 6
```

---

### 三項演算子で条件分岐を1行にまとめる

`condition ? valueIfTrue : valueIfFalse` の形で、`if-else` と同等の処理を1行で書ける。

```swift
// if-else で書く場合
if n >= 0 { return n } else { return -n }

// 三項演算子
return n >= 0 ? n : -n
```

シンプルな条件分岐には三項演算子が読みやすい。ネストが深くなる場合は `if-else` の方が明快。
