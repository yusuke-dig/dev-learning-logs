## 今日学んだこと

### switch で範囲マッチ

```swift
switch score {
case 90...100: return "A"
case 70..<90:  return "B"
default:       return "D"
}
```

`...`（閉区間）と `..<`（半開区間）を使い分ける。Swift の `switch` は `break` 不要で全ケース網羅が必須。

### switch + where で条件付きマッチ

```swift
case let x where x % 3 == 0 && x % 5 == 0:
```

`case let` でバインドして `where` で追加条件を指定できる。応用的な使い方。

### for-in + guard で範囲ループの罠を回避

```swift
guard n != 0 else { return 1 }
for i in 1...n { result *= i }
```

`n == 0` のとき `1...0` は実行時クラッシュになるため `guard` で早期リターンが必要。
