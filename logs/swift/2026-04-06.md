## 今日学んだこと
- `ContentUnavailableView`の使い方を学んだ。  
ContentUnavailableViewは、データがない時に簡単に表示できるView。アイコンとテキストを指定するだけで、データがないことをユーザーに伝えることができる。  
ios17で追加されたView。

- `textSelection(.enabled)`を使うと、テキストを選択できるようになる。
SwiftUIのデフォルトではTextは選択できないが、textSelection(.enabled)を指定することで、ユーザーがテキストを選択してコピーできるようになる。

## 実装したこと
- SwiftTutorialでoverlayと組み合わせて、データがないときにContentUnavailableViewを表示した。

```swift
.overlay {
    if moments.isEmpty {
        ContentUnavailableView {
            Label("No moments yet!", systemImage: "exclamationmark.circle.fill")
        } description: {
            Text("Post a note or photo to start filling this space with gratitude.")
        }
    }
}
```

- `textSelection(.enabled)`でメモの内容を選択できるようにした。
```swift
Text(moment.note)
    .textSelection(.enabled)
```
