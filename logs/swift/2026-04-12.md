### `StreakCalculator` のロジック

連続記録日数（ストリーク）を計算する処理。

```swift
let daysAgoArray = moments
    .reversed()              // 新しい順に並び替え
    .map(\.timestamp)        // [Moment] → [Date]
    .map { calendar.dateComponents([.day], from: $0, to: endOfToday) }  // [Date] → [DateComponents]
    .compactMap { $0.day }   // [DateComponents] → [Int]
```

---

### `map` と `compactMap` の件数の変化

```
map         → 件数は変わらない（変換だけ）
compactMap  → nilを除去するので件数が減る可能性あり
filter      → 条件に合わないものを除去するので件数が減る
flatMap     → 入れ子を平坦化するので件数が増える
```

---

### `map(\.timestamp)` とは

KeyPathを使ったプロパティの取り出し。

```swift
moments.map(\.timestamp)
// = moments.map { $0.timestamp }
// [Moment] → [Date]（timestampだけ取り出した新しい配列）
```

TypeScriptの `moments.map(m => m.timestamp)` と同じ概念。Swiftは KeyPath（`\.timestamp`）でより短く書ける。

---

### `dateComponents([.day], from:to:)` とは

2つの日付の差分を計算するメソッド。

```swift
calendar.dateComponents([.day], from: $0, to: endOfToday)
//                       ^^^^   ^^^^      ^^^^^^^^^^^
//                       日数   起点       終点
```

- `from` と `to` の2点が必要（差分は2点ないと計算できない）
- `[.day]` を指定することで時間を無視し**カレンダー上の日数差**だけを計算する
- 24時間ベースではなく**00:00:00を境に別日判定**される

```
2026-04-10 23:59:59 → 1日前
2026-04-11 00:00:00 → 0日前（今日）
```

- `Date` 同士を直接引き算できないため `DateComponents` を経由する必要がある

---

### `$0` とは

クロージャの最初の引数の省略記法。

```swift
// 省略形
.map { calendar.dateComponents([.day], from: $0, to: endOfToday) }

// 展開した形（同じ意味）
.map { date in calendar.dateComponents([.day], from: date, to: endOfToday) }
```

引数が複数あるときは `$0` `$1` `$2`... と対応する。

### ストリーク判定ロジック

```swift
var streak = 0
for daysAgo in daysAgoArray {
    if daysAgo == streak {
        streak += 1
    }
}
```

`streak` が「次に期待する日数」になっている。

```
streak=0 → 「今日（0日前）」を期待
streak=1 → 「昨日（1日前）」を期待
streak=2 → 「一昨日（2日前）」を期待
```

連続が途切れた瞬間に `daysAgo == streak` が不一致になり、streakが増えなくなる。

### `LazyVStack` とは

画面に表示されていない要素を描画しないようにするVStack。

```swift
// VStack → 全要素を一度に描画（1000件あれば1000件全部）
// LazyVStack → 画面に見える分だけ描画（パフォーマンス向上）
```

---

### `pinnedViews: .sectionHeaders` とは

スクロールしてもセクションヘッダーを上部に固定するオプション。

```swift
LazyVStack(spacing: 8, pinnedViews: .sectionHeaders) {
    Section {
        pathItems       // コンテンツ
    } header: {
        streakHeader    // スクロールしても固定されるヘッダー
    }
}
```

`LazyVStack` + `Section` + `pinnedViews` の3つセットで「固定ヘッダー＋遅延描画」が実現できる。

---

### `@ViewBuilder` とは

```swift
@ViewBuilder private var streakHeader: some View {
    let streak = StreakCalculator().calculateStreak(for: moments)
    if streak > 0 {
        HStack { ... }
    }
}
```

プロパティやメソッドの中で `if` などの条件分岐を使いながらViewを返せるようにするマクロ。`body` は最初から `@ViewBuilder` が付いているが、独自のプロパティで条件分岐を使うときは明示的に付ける必要がある。
